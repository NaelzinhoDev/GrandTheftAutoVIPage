# 🎮 GTA VI Access

Landing Page inspirada no lançamento de GTA VI, desenvolvida para praticar conceitos de React e Tailwind CSS.

O projeto simula uma página de cadastro para uma lista VIP de pré-venda, permitindo que usuários se registrem para receber novidades exclusivas sobre o lançamento do jogo.

## 🚀 Tecnologias Utilizadas

- React.js
- Vite
- Tailwind CSS
- JavaScript (ES6+)
- HTML5
- CSS3

---

## ✨ Funcionalidades

- Hero Section inspirada na identidade visual do GTA VI
- Layout responsivo para desktop
- Navbar moderna com efeito glassmorphism
- Formulário de cadastro utilizando React Hooks
- Cards de benefícios
- Efeitos de hover e transições suaves
- Interface moderna utilizando Tailwind CSS

---

## 📚 Conceitos Praticados

Durante o desenvolvimento deste projeto foram aplicados conceitos importantes de Front-end:

- Componentização com React
- Props e JSX
- React Hooks (useState)
- Organização de pastas
- Responsividade
- Estilização com Tailwind CSS
- Eventos e formulários
- Manipulação de estado

## ⚙️ Instalação

Clone o projeto:

```bash
git clone https://github.com/SEU-USUARIO/GrandTheftAutoVI-Access.git
```

Entre na pasta:

```bash
cd GrandTheftAutoVI-Access
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```
